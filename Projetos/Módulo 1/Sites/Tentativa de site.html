REVISANDO TUDO QUE EU APRENDI

Como criar paragrafos?

Para fazermos isso usamos a tag p que fica entre os simbulos menor < e maior >. Desse jeito: <p>

Como colocar imagens em um site?

Primeiro, precisamos de uma imagem, em seguida escrevemos a tag <img src="aqui será o nome igual a do arquivo da imagem. Ex:. planta.png/fotodocavalo.jpg">

LINHAS HORIZONTAIS

Para criarmos uma linha horizontal: usamos a tag <hr>. O que acontecerá é que vai criar uma linha horizontal onde ficará entre as linhas

QUEBRA LINHA

Para quebrar linhas como as de um poema usamos a tag br que como todas as outras, fica entre os simbulos especiais: < >.

ou posso dizer frank
